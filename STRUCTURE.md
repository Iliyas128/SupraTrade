# Структура проекта SNG Group

## Обзор

Проект представляет собой веб-сайт компании SNG Group KZ — поставщика металлопроката, химических реактивов, медицинского и промышленного оборудования в Казахстане и СНГ. Проект построен на React 18 с использованием TypeScript, Vite, Tailwind CSS и компонентов shadcn/ui.

## Технологический стек

### Основные технологии
- **React 18.3.1** — библиотека для построения пользовательского интерфейса
- **TypeScript 5.8.3** — типизированный JavaScript
- **Vite 5.4.19** — инструмент сборки и разработки
- **Tailwind CSS 3.4.17** — utility-first CSS фреймворк
- **React Router DOM 6.30.1** — маршрутизация в React приложении

### UI библиотеки
- **Radix UI** — набор доступных компонентов (accordion, dialog, dropdown, toast и др.)
- **shadcn/ui** — коллекция переиспользуемых UI компонентов на базе Radix UI
- **Lucide React** — библиотека иконок
- **Sonner** — система уведомлений (toast)
- **Embla Carousel** — карусель для слайдеров

## Описание ключевых компонентов

### App.tsx
Корневой компонент приложения, который:
- Настраивает провайдеры (TooltipProvider для подсказок)
- Инициализирует систему уведомлений (Toaster, Sonner)
- Настраивает маршрутизацию (React Router)
- Определяет основные маршруты приложения

### pages/Index.tsx
Главная страница сайта, которая объединяет все секции:
- TopBar — верхняя панель с контактами
- Header — шапка с навигацией
- HeroSlider — главный слайдер
- ProductCategories — категории продукции
- Directions — направления деятельности
- Advantages — преимущества
- Services — услуги
- Stats — статистика
- AboutSection — о компании
- Reviews — отзывы
- News — новости
- Partners — партнеры
- Licenses — лицензии
- ContactCTA — призыв к действию
- Footer — подвал
- FloatingButtons — плавающие кнопки
- ChatWidget — виджет чата
- CallbackModal — модальное окно обратной связи

### pages/Catalog.tsx
- Герой, статистика и форма поиска по товарам
- 6 направлений с подсекциями и ссылками (`/catalog/:direction`, `/catalog/:direction/:subcategory`)
- 15 случайных карточек товаров (маленькое фото + описание) с ссылкой на детальную страницу продукта
- Поиск по API (`/api/products/search`) с выводом до 15 результатов

### pages/CatalogCategory.tsx
- Динамическая маршрутная страница для направлений/подсекций
- Загружает категории с backend (`/api/categories`) и товары по slug (`/api/products/category/:slug`)
- Предусмотрена кнопка «Показать ещё» (пагинация по 15 шт.)
- Боковое меню с активным направлением и подсекциями

### pages/CatalogProduct.tsx
- Детальная карточка товара по полному пути (`/catalog/slug/.../product`)
- Запрос к API `/api/products/by-url/<path>`
- Галерея изображений (1-4 фото), характеристики, теги, CTA для запроса КП

### components/layout/

#### Header.tsx
Шапка сайта с:
- Логотипом компании
- Навигационным меню
- Выпадающим каталогом товаров
- Кнопкой обратного звонка
- Мобильным меню

#### Footer.tsx
Подвал сайта с:
- Информацией о компании
- Ссылками на разделы
- Контактной информацией
- Социальными сетями

### components/sections/

#### HeroSlider.tsx
Главный слайдер с изображениями и призывами к действию. Использует Embla Carousel для навигации.

#### ProductCategories.tsx
Отображает категории продукции с изображениями и ссылками.

#### CatalogDropdown.tsx
Выпадающее меню каталога с категориями и подкатегориями товаров. Данные берутся из `catalogData.ts`.

### components/ui/
Коллекция переиспользуемых UI компонентов на базе Radix UI и Tailwind CSS. Компоненты следуют принципам shadcn/ui:
- Доступность (accessibility)
- Кастомизируемость через props
- Типизация TypeScript

### data/catalogData.ts
Структурированные данные каталога товаров:
- Категории (металлопрокат, химические реактивы, оборудование)
- Группы товаров внутри категорий
- Подкатегории с ссылками

## Стилизация

### Tailwind CSS
Проект использует Tailwind CSS с кастомной конфигурацией:
- Кастомные цвета (primary, secondary, accent и др.)
- Кастомные тени (shadow-custom-*)
- Анимации (accordion, slide-left)
- Адаптивные breakpoints

### CSS переменные
Цвета определяются через CSS переменные в `index.css` для поддержки темной темы (хотя в текущей версии используется только светлая тема).

## Маршрутизация

Приложение использует React Router DOM для навигации:
- `/` — главная страница
- `/catalog` — главная страница каталога (направления + 15 товаров)
- `/catalog/:primarySlug` — страница направления
- `/catalog/:primarySlug/:subSlug` — подсекция направления
- `/catalog/:primarySlug/:subSlug/:rest*` — детальная карточка товара (глубина категорий произвольная)
- `*` — страница 404 для всех остальных маршрутов

## Сборка и разработка

### Скрипты (package.json)
- `npm run dev` — запуск dev-сервера на порту 8080
- `npm run build` — сборка production версии
- `npm run build:dev` — сборка development версии
- `npm run lint` — проверка кода линтером
- `npm run preview` — предпросмотр production сборки

### Конфигурация Vite
- Порт: 8080
- Host: `::` (все интерфейсы)
- Алиас `@` для `./src`
- Плагин React SWC для быстрой компиляции
- Lovable Tagger (только в development режиме)

## Удаленные зависимости

Следующие библиотеки были удалены как неиспользуемые:
- `@tanstack/react-query` — не использовался (нет API запросов)
- `recharts` — компонент chart не использовался
- `react-day-picker` — компонент calendar не использовался
- `cmdk` — компонент command не использовался
- `vaul` — компонент drawer не использовался
- `input-otp` — компонент input-otp не использовался
- `react-resizable-panels` — компонент resizable не использовался
- `react-hook-form` и `@hookform/resolvers` — не использовались (формы на useState)
- `zod` — использовался только с react-hook-form
- `date-fns` — не использовался
- `next-themes` — заменен на простую реализацию (тема "light")
